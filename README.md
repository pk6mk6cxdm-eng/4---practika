# 4---practika
6-вариант
Клиенттің аты, шот нөмірі және бастапқы балансы  енгізіледі, баланс жабық өріс болып есептеледі, ақша қосу немесе алу әрекетінен кейін соңғы баланс экранға шығарылады.

6-вариант
Сиқыршы ойыны жасалады, ойыншының мана және күш көрсеткіштері енгізіледі, мана жабық болады, сиқыр қолданғаннан кейін қалған мана мен нәтиже экранға шығарылады.
